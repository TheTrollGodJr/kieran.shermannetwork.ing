Daily Photos

Picture Time

This is a computer science project that I did for my high school capstone project. Starting in 2020, I took a photo of myself everyday so that it could be compiled into a time lapse video so you can see me over several years. I wrote a program in python that takes each photo in order from when they were taken and uses facial recognition and math to align the faces in each photo in a video for smooth transitioning. I later turned it into a website where I can upload photos everyday and have them automatically processed and updated. I also developed an API to send and receive processed photos and videos between clients and the server. I started writing the code for this project in 2023 and plan on having it completely finished by January 2025.
